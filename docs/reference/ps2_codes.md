# PS/2 Scan Code Reference

Reference for PS/2 scan code set 1 used by the keyboard driver.

## Make/Break Codes

Each key press sends a **make code**. Each key release sends a **break code** = make code + 0x80.

## Scan Code Set 1 (Make Codes)

| Key | Make | Break | Key | Make | Break |
|-----|------|-------|-----|------|-------|
| Escape | 0x01 | 0x81 | Tab | 0x0F | 0x8F |
| 1 (!) | 0x02 | 0x82 | q | 0x10 | 0x90 |
| 2 (@) | 0x03 | 0x83 | w | 0x11 | 0x91 |
| 3 (#) | 0x04 | 0x84 | e | 0x12 | 0x92 |
| 4 ($) | 0x05 | 0x85 | r | 0x13 | 0x93 |
| 5 (%) | 0x06 | 0x86 | t | 0x14 | 0x94 |
| 6 (^) | 0x07 | 0x87 | y | 0x15 | 0x95 |
| 7 (&) | 0x08 | 0x88 | u | 0x16 | 0x96 |
| 8 (*) | 0x09 | 0x89 | i | 0x17 | 0x97 |
| 9 (() | 0x0A | 0x8A | o | 0x18 | 0x98 |
| 0 ()) | 0x0B | 0x8B | p | 0x19 | 0x99 |
| - (_) | 0x0C | 0x8C | [ { | 0x1A | 0x9A |
| = (+) | 0x0D | 0x8D | ] } | 0x1B | 0x9B |
| Backspace | 0x0E | 0x8E | Enter | 0x1C | 0x9C |

## Keyboard Commands

| Command | Byte | Description |
|---------|------|-------------|
| Set LEDs | 0xED | Followed by LED state byte |
| Echo | 0xEE | Returns 0xEE (diagnostic) |
| Set Scancode Set | 0xF0 | Followed by set number (1, 2, 3) |
| Enable | 0xF4 | Clears buffer, starts scanning |
| Disable | 0xF5 | Stops scanning, resets |
| Reset | 0xFF | Hardware reset |
