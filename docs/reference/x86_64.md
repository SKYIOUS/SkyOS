# x86_64 Architecture Reference

This page documents the x86_64 features used by SkyOS.

## Registers

### General Purpose Registers

| Register | Purpose in SkyOS |
|----------|-----------------|
| RAX | Syscall number, return value |
| RBX | Callee-saved |
| RCX | 4th syscall argument |
| RDX | 3rd syscall argument |
| RSI | 2nd syscall argument |
| RDI | 1st syscall argument |
| R8 | 5th syscall argument |
| R9 | 6th syscall argument |
| R10 | 4th syscall argument (clobbered by syscall) |
| R11 | RFLAGS (clobbered by syscall) |
| R12-R15 | Callee-saved |
| RBP | Frame pointer (optional) |
| RSP | Stack pointer |

### System Registers

| Register | Purpose |
|----------|---------|
| CR0 | Protected mode, paging, FPU control |
| CR2 | Page fault linear address |
| CR3 | Page table base address (PML4) |
| CR4 | PAE, PGE, SMEP, SMAP, FSGSBASE |
| CR8 | Task priority register (APIC) |
| EFER | SYSCALL enable, NX bit, Long mode |

### Segment Registers

| Register | Purpose |
|----------|---------|
| CS | Kernel code (ring 0) or userspace code (ring 3) |
| SS | Stack segment |
| DS/ES | General data (unused in long mode) |
| FS | Thread-local storage base |
| GS | Kernel per-CPU data base |

## Instructions

### Privileged Instructions

| Instruction | Purpose |
|-------------|---------|
| `syscall` | Enter kernel from userspace |
| `sysret` | Return from kernel to userspace |
| `swapgs` | Swap GS base (kernel entry/exit) |
| `invlpg` | Invalidate a TLB entry |
| `hlt` | Halt CPU (idle loop) |
| `cli`/`sti` | Disable/enable interrupts |
| `lgdt` | Load GDT |
| `lidt` | Load IDT |
| `mov cr3` | Switch page tables |

### Atomic Instructions

| Instruction | Purpose |
|-------------|---------|
| `lock cmpxchg` | Atomic compare-and-swap |
| `lock xadd` | Atomic fetch-and-add |
| `lock inc/dec` | Atomic increment/decrement |
| `mfence` | Memory barrier (store) |
| `lfence` | Memory barrier (load) |
| `sfence` | Store fence |
